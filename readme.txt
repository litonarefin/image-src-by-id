Give Image URL with different size Options by Image ID.


== Description ==
This is very Simple and very Useful Plugin. If you redesign/re-Develop your website or any external plugin used Image ID for extracting on WordPress Editor. 
Uninstalling/Bugs may occur your Images Broken with only Image ID. 
There's no solution to get the Image Source and fix your Post/Page if you're not Developer/Technical Person. 
By default WordPress Editor Searches Images by Title not ID. 
This Plugin will help you just Place the Image ID and Get the exact Image Size with Preview Image also. 

Then you can easily search on Media Popup for the specific Image and fix your problems. You don't have to hire any freelancer to fix this issue.

== Installation ==
*  From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'image-src-id.zip' file and upload it. Or,

*  Extract the zip file, copy and paste to the 'wp-conten/plugins/' directory of your WordPress Installation.

* Activate the plugin through the 'Plugins' menu in WordPress

* "Image Src by ID" Menu will be found under Setting Menu.
